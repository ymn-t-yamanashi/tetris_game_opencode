# TetrisGameOpencode（オープンコードでテトリスを作るプロジェクト）

## このリポジトリについて
本リポジトリは、Elixir と Phoenix を使ってブラウザ上で動くテトリスゲームを構築するサンプルです。 
***現在作成中でまだ動きません***

## 動かす手順（日本語）
1. **依存関係のインストール**
   ```bash
   mix setup
   ```
2. **サーバー起動**
   ```bash
   mix phx.server
   ```
3. ブラウザで `http://localhost:4000` を開くとゲームが表示されます。

## 使える機能
- ブロックの落下・回転・消去
- 行を揃えるごとにスコアアップ
- ゲームオーバー時の再スタートボタン
- スクロールやズームなど、ブラウザサイズに合わせて自動調整
