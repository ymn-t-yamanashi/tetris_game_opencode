# テトリス

## ゲームの流れ
1. 画面にブロック（テトリミノ）が落ちてきます。
2. 左右キーでブロックを動かし、上矢印で回転させます。
3. ブロックが下まで届くと固定され、新しいブロックが出現します。
4. 横に並んだブロックが1列揃うと、その行が消えて得点が入ります。
5. ブロックが画面の上まで積み重なってしまうとゲームオーバーです。

## 目的
- 一番長く生き残ること、または高いスコアを取ることが目標です。

## 用語集（小学生向け）
- **テトリミノ**：四つのブロックでできた形。例えば「L字型」や「I字型」。
- **ライン消し**：横に並んだブロックを一列揃えると、その行が消えて得点になること。
- **ハイスコア**：これまでの最高スコア。

## ルールまとめ（簡単）
- ブロックは落ちる速度で動きます。早くなるほど難しくなります。
- 上にブロックが止まってしまうとゲームオーバーです。
- 「スペースキー」を押すとブロックを下へ急速に落とせます。

## 詳細ルール（Elixirで実装する前提）
1. **テトリミノの種類**
   - I、O、T、S、Z、J、L の7種類があります。各ブロックは4つのセルから構成されます。
   - それぞれの回転パターンを表す座標配列（相対座標）を Elixir のリストで保持します。

2. **フィールドサイズ**
   - フィールドは横10セル、縦20セルのグリッドです。`GameField` モジュールで `@width 10`, `@height 20` を定義し、配列で空き/埋まりを管理します。

3. **落下速度**
   - 初期は1秒に1行落ちますが、レベルが上がるごとに速くなります。レベルは消したライン数で決め、`GameState.level/0` を更新します。
   - 例：`level = div(lines_cleared, 10)`

4. **ライン消し**
   - 完全に埋まった行を検出し、配列から削除して上のブロックを1つずつ下へ移動します。
   - 消した行数に応じてスコアが増加。`Score.calculate/2` で `lines * (level + 1) * 100` の計算式を使用します。

5. **回転と壁クイック**
   - 回転は「SRS（Super Rotation System）」に従い、最大9回のオフセット試行を行います。Elixir では `Enum.reduce_while/3` を使って最適位置を探します。

6. **ゲームオーバー判定**
   - 新しいブロックが生成されるときにフィールド上部に既にセルが存在している場合、ゲームオーバー。`GameState.game_over?/0` で判定し、フラグを立てます。

7. **入力処理**（Elixir LiveView）
   - キーボードイベントは `handle_event/3` で受け取り、`GameController.move_left/1`, `move_right/1`, `rotate/1`, `hard_drop/1` を呼び出します。
   - それぞれの関数はフィールド境界と衝突判定を行い、状態を更新します。

8. **スコア保存**
   - 高得点は `UserProfile` テーブルに保存し、ログインユーザーごとに保持します。`Repo.insert_or_update/1` で永続化。

これらのルールを元に Elixir でゲームロジックを実装すれば、小学生でも楽しく遊べるテトリスが完成します！
